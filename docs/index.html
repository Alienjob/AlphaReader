AlphaReader

Georgia language learning solution

A mobile application that allows you to transliterate individual sounds into symbols of the Georgian alphabet. It helps to learn how to read the text in Georgian in an easy and convenient way. The application is developed using a clean architecture, state management is done using BLOC
